# -ca-finance-toolkit
        An open-source collection of finance, accounting, and tax utilities, calculators, templates, and automation tools for students and professionals.
## About
CA Finance Toolkit aims to simplify routine accounting and taxation tasks through free and open-source tools.

## Planned Features

- GST Interest Calculator
- GST Late Fee Calculator
- Financial Ratio Calculator
- Working Capital Calculator
- EMI Calculator
- Income Tax Utilities
- Excel Templates
- Power BI Dashboard Templates

## Why this project?

Finance professionals often rely on multiple websites or paid software for simple calculations. This project brings practical tools together in one open-source repository that anyone can use, improve, and contribute to.
